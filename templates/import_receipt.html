Upload Receipt File

Upload a JSON or CSV file from your POS system to generate an ETR-compliant receipt with QR code.

How It Works

  1. Export receipt data from your POS system as JSON or CSV
  2. Upload the file using the form
  3. System extracts total amount and calculates VAT
  4. Generates ETR-compliant receipt with QR code
  5. Print or save the receipt for KRA compliance

Supported Fields:

  • total, total_amount, amount
  • grand_total, final_amount
  • Nested objects with summary.total

Sample JSON Format

{
    "total": 1250.00,
    "items": [
        {"name": "Product 1", "price": 500, "quantity": 1},
        {"name": "Product 2", "price": 750, "quantity": 1}
    ],
    "customer": "John Doe",
    "payment_method": "Cash"
}
